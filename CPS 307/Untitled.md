For Loop
for(initialize your variable;the condition;increment){

}

```Javascript
array = [1,2,3,4,5]
for(var i=0 ; i<array.length;i++){
	console.log(array[i])
}
```

Do while 

do{
code
}while(condition)

```Javascript
array = [1,2,3,4,5]
var i = 0;
do{
console.log(array[i])
i++;
}while(i<array.length)
```
